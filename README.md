# 圣经知识竞赛 / Bible Knowledge Competition

这是一个静态网页版本的圣经知识竞赛应用，当前以中文为默认基线，并支持中英切换。

## 当前功能

- 设置页支持比赛标题、分组数量、每题计时
- 支持中英切换，中文默认
- 题库内容本身也是中英双语
- 题目通过独立配置文件读取，不写死在业务逻辑里
- 进入答题页后，各组可以同时作答
- 同一轮中，每组会拿到不同的题目
- 答题页改为横版分组面板
- 每轮结束后按答对题数累计积分，并显示积分榜
- 本地使用 `localStorage` 保存当前设置与比赛状态

## 使用方式

1. 打开 `index.html`
2. 设置比赛标题、分组数量、每题计时，并填写各组名称
3. 点击“开始答题”
4. 进入 `quiz.html` 后，各组在同一轮同时答题

## 文件结构

- `index.html`：设置页
- `quiz.html`：横版答题页
- `shared.js`：公共语言、文本和工具函数
- `setup.js`：设置页逻辑
- `quiz.js`：答题页逻辑
- `styles.css`：共享样式
- `data/questions.js`：题库配置文件

## 题库配置

后期如果要自己维护题目，只需要编辑 `data/questions.js`。

- 这个文件现在是独立配置对象
- 页面逻辑会从配置中读取题目
- 保持字段名不变即可继续正常使用
